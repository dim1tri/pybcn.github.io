Python Programming Introduction Course (I)

Time:
11:15
Room:
Sala Multimèdia
Abstract:

A beginner to Python programming course.

NOTICE: This workshop will be given in Spanish or Catalan, depending on audience. REQUIREMENTS: You don't need laptop for this workshop, but you may need a USB stick to save your files. If you bring your laptop, you will need to get Python 2.7, Sublime Text 2.

Speaker:
Eloi Puertas
Speaker background:

Eloi Puertas is a Computing Science Professor at Facultat de Matemàtiques i Informàtica of Universitat de Barcelona, and a collaborator at PyBCN.
